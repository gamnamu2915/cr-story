# 소재 추출: 게시판

## 화두
미디어 리터러시 수업 과제로 분석한 '문제 기사'가 같은 반 학생의 아버지가 쓴 기사라면?

## 컨셉트
고등학생 서준이 미디어 리터러시 수업에서 CR-Check 도구로 기사를 분석한다. 우수 과제로 선정되어 복도 게시판에 게시된다. 그런데 분석 대상 기사를 쓴 기자가 같은 반 도윤의 아버지(박현우 기자)라는 사실을 뒤늦게 알게 된다.

## 핵심 장면
1. **분석 수업**: 서준이 CR-Check 도구로 "사교육비 역대 최고" 기사를 분석하며 익명 취재원, 균형성 결여 등의 문제를 발견
2. **게시판 게시**: 우수 과제로 선정되어 복도 게시판에 공개적으로 전시됨
3. **발견의 순간**: 도윤이 게시판에서 자기 아버지 이름(박현우)을 발견
4. **상담 주간 조우**: 복도에서 서준과 박현우 기자가 마주침
5. **운동장 대화**: 도윤이 서준에게 아버지의 입장 전달 - "마감 1시간, 취재원은 익명 요청, 교육부는 응답 없음. 구조가 그렇다"
6. **마지막 성찰**: 게시판 앞에서 서준이 "분석은 맞았다. 그런데 맞는 것만으로 충분한가"라고 자문

## 인물
- **서준**: 주인공. 미디어 리터러시 수업에서 기사를 분석하는 고등학생. 처음엔 과제로만 생각했지만 분석 과정에서 빠져듦
- **도윤**: 서준의 같은 반 학생. 조용하고 눈치 빠른 성격. 박현우 기자의 아들. 아버지의 기사가 문제 있다는 걸 어렴풋이 알고 있었음
- **박현우**: 한성일보 기자. 도윤의 아버지. 마감과 데스크의 압박 속에서 윤리 규범과 타협하며 기사를 쓰는 현실적 기자
- **김예은**: 미디어 리터러시 수업 선생님. 학생들에게 비판적 사고를 가르침

## 주제
- 윤리적 정확성과 인간적 복잡성의 긴장
- "분석은 맞았다. 그런데 맞는 것만으로 충분한가?"
- 언론 윤리 규범 vs 현장의 구조적 압박
- 공적 비판과 사적 관계의 충돌

## 특이사항
- 원본 스토리 구조: 6개 섹션으로 나뉘어 점진적으로 갈등 전개
- 서준의 시점이 중심이지만, 도윤의 내면과 박현우의 현실도 간접적으로 드러남
- 게시판이라는 "공적 공간"이 핵심 무대
- 마지막은 명확한 해결 없이 질문으로 끝남 - "다음에는 어떻게 할까. 다만 이전과 같지는 않을 것이다"
