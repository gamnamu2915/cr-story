# CR-Story 터치업 가이드라인

> 이 문서는 CR-Story 파이프라인의 터치업 단계에서 문장 리듬을 개선하기 위한 규칙과 예시를 담고 있습니다.
> 한국 중견작가(이청준, 오정희, 박완서, 윤흥길)의 문체 분석을 바탕으로 작성되었습니다.

---

## 핵심 원칙

**"단문은 예외, 복문이 기본"**

좋은 한국 단편소설은 중-장문의 호흡을 기본으로 하며, 단문은 특별한 순간에만 사용합니다.

---

## 규칙 1: 단문 연속 금지 (3문장 이상)

### 정의
- **단문**: 주어+서술어만으로 이루어진 15자 이하의 문장
- **연속**: 3개 이상의 단문이 문단 내에서 이어지는 경우

### 금지 패턴
```
❌ 메일함을 닫았다. 강의 시간이었다. 강의실로 갔다.
❌ 창을 닫았다. 화면을 오래 봤다. 잠들지 못했다.
❌ 다시 자리에 앉았다. 답장 버튼을 눌렀다. 흰 막대가 깜빡거렸다.
```

### 개선 방법
1. **접속사 연결**: ~고, ~며, ~지만 등으로 연결
2. **종속절 활용**: ~할 때, ~하면서, ~한 채로
3. **이유/결과 연결**: ~해서, ~하니, ~하자

---

## 규칙 2: 단문 허용 조건

단문은 다음 상황에서만 1-2문장 사용합니다:

| 허용 상황 | 예시 | 효과 |
|----------|------|------|
| **절정/충격** | "전화가 끊겼다." | 긴장감 극대화 |
| **대화 직후 반응** | "고개를 끄덕였다." | 간결한 호응 |
| **결말부 여운** | "그는 답장을 보냈다." | 여운 남기기 |
| **내면 전환점** | "그래서 갔다." | 결단의 순간 |

### 주의
- 허용 조건에서도 **연속 2문장을 넘기지 않음**
- 단문 후에는 반드시 중-장문으로 호흡 회복

---

## 규칙 3: 내면 독백은 흐름으로

### 금지 패턴
```
❌ 생각했다. 맞는 말이다. 하지만 찝찝했다.
❌ 연구와 현실 사이. 논문 속의 "주목할 만한" 것과, 실제로 주목하는 것 사이.
```

### 개선 패턴
```
✅ 태윤 말이 맞다고 생각했지만, 어딘가 찝찝한 구석이 남았다.
✅ 연구와 현실 사이의 거리가, 논문 속에서 "주목할 만하다"고 쓴 것들을 실제로는 주목하지 않는 자신의 모습이 떠올랐다.
```

### 원칙
- 사유의 흐름은 끊기지 않고 이어져야 함
- 파편적 명사구 나열 지양
- 생각의 연결고리를 문장 안에 포함

---

## 규칙 4: 장면 전환 후 호흡 회복

### `---` 구분선 직후 규칙
1. 첫 문장에서 **시간+공간** 모두 설정
2. 중-장문(30자 이상)으로 시작
3. 단문으로 시작 금지

### 금지 패턴
```
---

❌ 토요일 오후.

❌ 금요일이었다. 메일함을 열었다.
```

### 개선 패턴
```
---

✅ 토요일 오후, 민준은 오랜 망설임 끝에 카페로 향했다.

✅ 금요일 오후 연구실에서 메일함을 열자, 새 메일이 하나 더 와 있었다.
```

---

## 규칙 5: 대화와 서술의 리듬

### 대화 전후 규칙
- 대화 직전: 행동/상황 묘사 (중문)
- 대화 직후: 짧은 반응 허용 (단문 1개)
- 반응 후: 다시 중-장문으로 복귀

### 예시
```
✅ 태윤이 계단을 오르며 말했다.
   "조용히 내 연구나 하는 게 맞아."
   고개를 끄덕였다. (← 단문 허용)
   태윤의 뒷모습을 보며 걷는 동안, 민준은 문득 10년 전 임용 면접을 떠올렸다. (← 호흡 회복)
```

## Before/After 예시 모음

### 예시 1: 행동 연속
```
[Before]
자리에서 일어나 창가로 갔다. 캠퍼스가 내려다보였다. 사회과학대학 5층. 
학생들이 걷고, 새들이 날고, 봄바람이 나무를 흔들었다.

[After]
자리에서 일어나 창가로 다가서자, 사회과학대학 5층에서 내려다보이는 캠퍼스가 
한눈에 들어왔다. 학생들이 걷고, 새들이 날고, 봄바람에 나무들이 흔들리는 
평화로운 오후였다.
```

### 예시 2: 메일 조작
```
[Before]
메일창을 닫았다. '임시저장 하시겠습니까?' 예.
임시저장함을 열었다. 거기에는 수십 개의 메일 초안들이 있었다.

[After]
메일창을 닫자 '임시저장 하시겠습니까?'라는 창이 떴고, 그는 '예'를 눌렀다.
임시저장함을 열어보니 수십 개의 메일 초안들이 쌓여 있었다.
```

### 예시 3: 정보 탐색
```
[Before]
제자의 이름을 검색창에 쳤다. SNS에서 찾았다. 프로필 사진: 정장 입은 청년.
소속: 모 일간지 사회부.

[After]
제자의 이름을 검색창에 치자 SNS 계정이 나왔다. 프로필 사진 속에는 
정장을 입은 청년이 있었고, 소속은 모 일간지 사회부라고 적혀 있었다.
```

### 예시 4: 밤 장면 전환
```
[Before]
창을 닫았다.
화면을 오래 봤다.
그날 밤, 그는 오래 잠들지 못했다.

[After]
창을 닫고 한동안 어두운 화면을 바라보았다. 
그날 밤, 그는 오래도록 잠들지 못했다.
```

### 예시 5: 장면 전환 (구분선 직후)
```
[Before]
---

토요일 오후.

[After]
---

토요일 오후, 민준은 결국 카페로 향했다.
```

### 예시 6: 내면 독백
```
[Before]
연구와 현실 사이.
논문 속의 "주목할 만한" 것과, 실제로 주목하는 것 사이.

[After]
연구와 현실 사이의 간극이 떠올랐다. 논문에서 "주목할 만하다"고 썼던 것들을 
정작 자신은 얼마나 주목하고 있었던가.
```

## 체크리스트 (터치업 전 확인)

- [ ] 단문이 3개 이상 연속되는 구간이 있는가?
- [ ] `---` 구분선 직후가 단문으로 시작하는가?
- [ ] 내면 독백이 파편적 명사구로 나열되어 있는가?
- [ ] 대화 직후 반응이 2문장 이상 단문으로 이어지는가?
- [ ] 절정/결말 외의 장소에서 불필요한 단문이 있는가?

## 레퍼런스 작가별 특징

| 작가 | 기본 호흡 | 단문 사용 | 참고할 점 |
|------|----------|----------|----------|
| 이청준 | 중-장문 | 체념/결단 순간만 | 내면과 행동의 자연스러운 연결 |
| 오정희 | 장문 | 거의 없음 | 감각 이미지의 연쇄 |
| 박완서 | 중문 | 감정 강조 시 | 대화와 서술의 교차 리듬 |
| 윤흥길 | 장문 | 대화 직후만 | 분위기 묘사의 지속성 |

## 적용 우선순위

1. **최우선**: 단문 3연속 이상 구간 수정
2. **높음**: 장면 전환 직후 단문 시작 수정
3. **중간**: 내면 독백 파편화 수정
4. **낮음**: 허용 범위 내 단문의 자연스러움 개선
