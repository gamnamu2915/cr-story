# 서사 기법 가이드 (NARRATIVE_GUIDE.md)

> **목적**: 한국 중견작가 단편소설에서 추출한 서사 원리
> **레퍼런스**: 이문열, 신경숙, 황석영, 오정희, 박완서, 윤흥길, 이청준 외
> **적용 대상**: 소설가A (05, 07, 09 단계)

---

## 1. 감정 곡선 설계

### 1.1 기본 구조

```
긴장도
  ^
  │            ↗ 위기2        ↗ 절정
  │          ↗    ↘        ↗     ↘
  │        ↗        ↘    ↗         ↘ 여운
  │      ↗ 위기1     ↘↗
  │    ↗
  │  ↗
  └───────────────────────────────────────→ 시간
     도입    전개    위기    절정    결말
    (10-15%) (25-30%) (30-35%) (15-20%) (10-15%)
```

### 1.2 구간별 특징

| 구간 | 분량 | 감정 상태 | 핵심 과제 | 주의사항 |
|------|------|----------|----------|----------|
| **도입** | 10-15% | 평온 → 불안 | 훅, 복선 심기 | 설명 과잉 금지 |
| **전개** | 25-30% | 불안 → 긴장 | 갈등 표면화 | 너무 빨리 격화 금지 |
| **위기** | 30-35% | 긴장 → 위기 | 점층적 고조 | **가장 힘 빠지기 쉬움** |
| **절정** | 15-20% | 위기 → 폭발 | 결정적 순간 | 과잉 설명 금지 |
| **결말** | 10-15% | 폭발 → 여운 | 열린 이미지 | 급한 마무리 금지 |

### 1.3 핵심 원칙

1. **긴장-이완의 반복**
   - 계속 긴장만 있으면 독자가 지침
   - 위기 후 잠시 숨 돌릴 틈, 그러나 곧 더 큰 위기

2. **점층적 상승**
   - 각 위기가 이전보다 반드시 더 큼
   - 위기1 < 위기2 < 절정

3. **가짜 해결**
   - 해결된 듯하다가 더 큰 문제 등장
   - 독자의 안도감을 배신

## 2. 중반 긴장 유지 5대 기법

### 2.1 새로운 장애물 투입

**원리**: 기존 갈등이 해소될 듯할 때 새로운 장애물 투입

**예시 (이문열 「우리들의 일그러진 영웅」)**:
```
[초반] 전학생 한병태, 엄석대의 권위에 반항
[중반 - 해결될 듯] 담임에게 호소하려 함
[새 장애물] 담임마저 엄석대 편 → 더 절망적
```

**적용법**:
- 주인공이 해결책을 찾을 때 그 해결책을 무력화
- 조력자가 배신하거나 무능함이 드러남
- 외부 환경이 악화됨

### 2.2 점층적 폭로

**원리**: 정보를 한 번에 주지 않고 조금씩 공개

**예시 (황석영 「아우를 위하여」)**:
```
[초반] 형이 동생에게 편지를 쓴다
[중반] 19년 전 사랑 이야기가 있었다 (궁금증)
[점점] 그 사랑의 정체가 조금씩 밝혀짐
[후반] 완전한 진실 공개
```

**적용법**:
- 핵심 정보를 3-4단계로 분산
- 각 폭로가 새로운 질문을 생성
- 독자가 "그래서?"를 계속 묻게 함

### 2.3 기대 뒤집기

**원리**: "될 것 같았는데 안 됐다"의 반복

**패턴**:
```
희망 → 좌절 → 더 큰 희망 → 더 큰 좌절 → 결정적 순간
```

**예시**:
```
[희망] 드디어 증거를 찾았다
[좌절] 증거가 위조된 것이었다
[더 큰 희망] 진짜 증거를 가진 사람을 만남
[더 큰 좌절] 그 사람이 입을 다물어버림
```

### 2.4 시간 압박

**원리**: 데드라인을 설정하여 긴장감 유지

**유형**:
- 물리적 시간: "내일 오전 10시까지"
- 기회의 창: "이번이 마지막 기회"
- 카운트다운: 점점 줄어드는 시간 표시

**예시 (테스트 작품 「답장」)**:
```
"읽음 표시. 오후 4시 32분."
"6시간 46분 경과."
"10시간 42분 남았다."
```

### 2.5 내적/외적 갈등 교차

**원리**: 한쪽이 잠잠할 때 다른 쪽 격화

```
[외적 갈등 격화] → [내적 갈등 잠시 잠잠]
[외적 갈등 소강] → [내적 갈등 격화]
```

**예시**:
```
[외적] 상사와의 충돌이 일단락
[내적] 그러나 자신의 비겁함에 대한 자괴감 폭발
[외적] 다시 상사가 더 큰 요구
[내적] 이제 어떻게 할 것인가? 선택의 기로
```

## 3. 결말 처리 기법

### 3.1 피해야 할 결말

| 안티패턴 | 예시 | 문제점 |
|----------|------|--------|
| 모든 해결 | "모든 갈등이 풀렸다" | 현실감 없음 |
| 직접 설명 | "그는 깨달았다" | 여운 없음 |
| 교훈 전달 | "이것이 삶의 진리다" | 설교적 |
| 대화 종결 | "그래, 그렇게 하자" | 여운 없음 |
| 새 사건 | 결말에 새 인물 등장 | 산만함 |

### 3.2 좋은 결말 4가지 유형

#### A. 열린 이미지

직접 설명 대신 이미지로 종결

**예시 (신경숙 「빈집」)**:
```
❌ "그는 그녀를 영원히 잃었다는 것을 깨달았다."

✅ "창밖으로 눈이 내리고 있었다. 
   그는 오래도록 그 자리에 앉아 있었다."
```

#### B. 순환 구조

처음과 끝이 연결되는 이미지

**예시**:
```
[도입] "아침마다 그는 같은 버스를 탔다."
...
[결말] "다음 날 아침, 그는 버스 정류장을 지나쳤다."
```

#### C. 질문으로 마무리

답을 주지 않고 질문만 남김

**예시**:
```
"그가 정말 옳은 선택을 한 것일까.
 아무도 모를 일이었다."
```

#### D. 감각으로 마무리

추상적 깨달음 대신 구체적 감각

**예시 (오정희 「동경」)**:
```
❌ "그녀는 깊은 상실감을 느꼈다."

✅ "차가운 바람이 볼을 스쳤다. 
   손끝이 시렸다."
```

### 3.3 결말 체크리스트

- [ ] 마지막 문장이 이미지/감각인가?
- [ ] 과잉 설명 없이 여운이 있는가?
- [ ] 독자가 스스로 의미를 찾을 공간이 있는가?
- [ ] 급하게 마무리하지 않았는가?
- [ ] 결말부 분량이 전체의 10-15%인가?

## 4. 대화와 서술의 배분

### 4.1 구간별 황금 비율

| 구간 | 대화 | 서술 | 이유 |
|------|------|------|------|
| 도입부 | 20-30% | 70-80% | 상황 설정, 분위기 조성 |
| 전개부 | 40-50% | 50-60% | 갈등이 대화로 표면화 |
| 위기부 | 50-60% | 40-50% | 대화로 충돌, 그러나 내면도 |
| 절정부 | 30-40% | 60-70% | 내면 묘사 강화 |
| 결말부 | 10-30% | 70-90% | 여운, 이미지, 성찰 |

### 4.2 핵심 규칙 3가지

#### 규칙 1: 대화 3회 연속 후 서술 단락

**원리**: 대화만 계속되면 독자가 멈춰 생각할 틈 없음

**Before (대화 과잉)**:
```
"정말이야?" 민수가 물었다.
"응, 진짜야."
"언제?"
"내일."
"알았어."
"고마워."
```

**After (서술 삽입)**:
```
"정말이야?" 민수가 물었다.
"응, 진짜야."

민수는 창밖을 바라보았다. 가로등 불빛이 빗방울에 부서지고 있었다. 
내일이라니. 그렇게 빨리 올 줄은 몰랐다.

"알았어." 겨우 그 말을 내뱉었다.
```

#### 규칙 2: 감정 고조 시 서술 강화

**원리**: 중요한 순간일수록 내면 묘사 비중 증가

**약한 처리**:
```
"나는 널 용서할 수 없어." 그녀가 말했다.
"미안해." 그가 대답했다.
```

**강한 처리**:
```
"나는 널 용서할 수 없어."

그녀의 목소리는 떨리지 않았다. 오히려 너무 평온해서 더 무서웠다. 
그는 그 눈빛을 마주할 수 없었다. 
십 년의 세월이 한순간에 무너지는 소리가 들리는 듯했다.

"미안해." 
그게 전부였다. 다른 말은 나오지 않았다.
```

#### 규칙 3: 결말부는 서술 중심

**원리**: 대화로 끝내지 않음

**피해야 할 결말**:
```
"그럼 이만." 
"응, 잘 가."
"다음에 또 보자."
"그래."

[끝]
```

**좋은 결말**:
```
"다음에 또 보자."

그는 대답하지 않았다. 그저 고개만 끄덕였다.
골목을 빠져나가는 그녀의 뒷모습이 가로등 아래로 사라졌다.
다음은 없을 것이다. 둘 다 알고 있었다.

[끝]
```

## 5. 레퍼런스 작품별 특징

### 5.1 이문열 「우리들의 일그러진 영웅」

**강점**:
- 회상 프레이밍 (30년 전 → 현재 시점)
- 권력 갈등의 점층적 고조
- 어린이 시점의 제한적 이해

**배울 점**:
- 새로운 장애물을 계속 투입
- 담임, 아버지 등 어른들도 해결책이 안 됨
- 기대의 연속적 좌절

### 5.2 신경숙 「빈집」

**강점**:
- 의식의 흐름 기법
- 현재와 회상의 자연스러운 교차
- 감각적 이미지로 결말

**배울 점**:
- 긴 호흡의 문장으로 내면 탐구
- 설명 대신 이미지
- 열린 결말의 여운

### 5.3 황석영 「아우를 위하여」

**강점**:
- 서간체(편지) 형식
- 과거 이야기의 점층적 폭로
- 따뜻한 회고 톤

**배울 점**:
- 정보를 조금씩 공개
- 독자의 궁금증 유지
- 과거와 현재의 교차

### 5.4 오정희 「동경」

**강점**:
- 감각적 묘사의 극치
- 내면의 미세한 변화 포착
- 압축된 문장

**배울 점**:
- 추상 대신 감각
- 여백의 힘
- 문장 리듬의 변화

### 5.5 박완서 「꿈꾸는 인큐베이터」

**강점**:
- 일상의 디테일
- 여성 화자의 솔직한 목소리
- 유머와 비애의 공존

**배울 점**:
- 구체적 사물로 감정 전달
- 독백의 자연스러움
- 복잡한 감정의 표현

## 6. 분할 작성 시 리마인더

### 6.1 1단계 완료 후 (도입-전개)

```
🔔 리마인더: 2단계 진입 전 확인

□ 훅이 작동했는가?
□ 갈등이 명확하게 설정되었는가?
□ 단문 비율 20% 이하인가?
□ 대화/서술 균형이 맞는가?

⚠️ 2단계는 가장 힘 빠지기 쉬운 구간입니다.
   긴장을 반드시 고조시키세요.
```

### 6.2 2단계 완료 후 (위기-절정)

```
🔔 리마인더: 3단계 진입 전 확인

□ 1단계보다 긴장이 높아졌는가?
□ 새로운 장애물/폭로가 있었는가?
□ 중간에 힘 빠진 구간이 없는가?
□ 절정을 향해 가고 있는가?

⚠️ 3단계에서 급하게 마무리하지 마세요.
   결말부는 전체의 10-15%입니다.
```

### 6.3 3단계 완료 후 (결말)

```
🔔 최종 점검

□ 결말이 급하지 않은가?
□ 마지막 문장이 이미지/감각인가?
□ 과잉 설명 없이 여운이 있는가?
□ 대화로 끝나지 않았는가?
```
